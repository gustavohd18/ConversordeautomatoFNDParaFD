Roda no terminal do linux utilizando o comando python3 e os proximos passos, caso utilize windows tenha
o python3 instalado como variavel global no sistema operacional e substitua todos os comandos por python em vez de 
python3:

python3 nomeCodigo.py "nomeArquivoLido.txt"

exemplo: python3 TF.py "automato.txt" ou no windows python TF.py "automato.txt"
